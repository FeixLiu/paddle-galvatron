------------------------------------  

The following table summarizes the currently supported ConvBERT models and their corresponding pretrained weights in PaddleNLP.  
For detailed model specifications, please refer to the corresponding links.

|                                                                                  |              | The ConvBERT base model.                                                         |  
+----------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------+

|``convbert-medium-small``                                                        | English      | 12-layer, 384-hidden,                                                            |
